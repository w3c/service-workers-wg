Service Workers Working Group

Our Mission

We provide methods to measure and improve aspects of application performance of user agent features and APIs.

The Co-Chairs of the Working Group are Jake Archibald and Matt Falkenhagen. The W3C Team Contact for the Service Workers Working Group is Yves Lafon.

We have a Service workers explainer. That's a good introduction for you.

What is Service Workers?

A service worker is a script that your browser runs in the background, separate from a web page, opening the door to features that don't need a web page or user interaction.
